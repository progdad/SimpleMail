About Simple Mail

The name of this web application speaks for itself meaning.

This is the noobiest mail service you've ever seen, but why not :)

    There are common actions you are able to do:

  • Send mails
  • Receive mails
  • Also you can paginate mails. Pagination is divided on 5 mails on each page
  • Read them and Delete them: If you want to delete any mail, it will be deleted only from your mails list !
    There is no way to delete a mail from user mails list you are chatting with.
  • Change your profile data: You can go to your profile page and change its data just clicking on your profile picture on navigation bar. And the coolest possibility is to change your profile picture. But a picture that you are uploading will be saved only for a one session, because of heroku.
  • View other users profiles: Each mail, no matter, received or sent, has a nickname field of user you are chatting with. You can click on his nickname or profile picture, and you will be redirected on his profile page.

This application was created by a junior python developer, who is a true supporter of open source code. Ofc this is just a pet-project, but I am certainly satisfied of the result !
So that i leave here this project github repository :)

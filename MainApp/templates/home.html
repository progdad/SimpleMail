About Simple Mail

The name of this web application speaks for itself meaning

This is a very simple service that is something between mail and social medias

    There are common rules and actions you can do:

  • At first you have to register your account here Otherwise you can't do anything, even visit users profiles.
  • Send, receive, delete and sort mails: When you delete mail, it is being deleted only from your account.
    Sorting mails displays matches in usernames(users you are chatting with) and titles.
  • Set your profile data and visit other users profiles. Check your profile here.
  • {% if request.user.is_authenticated %} Also you can test this app API here. {% else %} This app has its own API, so after being registered you can simply use it. {% endif %}

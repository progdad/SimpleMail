Simple Mail

Simple Mail is a basic web application created as a personal project to practice Python programming skills. It's a minimalistic platform that combines email-like functionality with a hint of social interaction.

    Features

  • User Registration To explore this app, you need to create an account. {% if not request.user.is_authenticated %} Sign Up Here! {% endif %}
    Without registration, you won't be able to view user profiles or send messages.
  • Email Management You can send, receive, delete, and sort your emails.
    Please note that deleting a message removes it from your account only. Sorting emails allows you to find messages by users and titles.
  • User profiles Customize your profile with your own information and visit other users' profiles to connect with fellow users.
